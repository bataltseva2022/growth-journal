# Growth Journal

Личное приложение для планирования задач, ведения дневника и отслеживания личного прогресса.

## Возможности

- Календарь
- Задачи
- Подзадачи
- Таймер
- Питомцы
- Темы оформления

## Планируется

- Статистика
- Heatmap
- Экспорт данных

## Стек

- React
- TypeScript
- Vite
- Tailwind CSS
